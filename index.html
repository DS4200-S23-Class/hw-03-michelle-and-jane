hw-03: Interactive Webpage, and Marks & Channels

Marks

  • area mark
    • Each of the states on the map represent an area mark with both height and width.
  • line mark
    • The bar at the top of the visualization represents a line mark showing the amount of democrat electoral votes versus republican electoral votes.

Channels

  • Position-both horizontal and vertical channel (spatial region) (identity channel)
    • Represents categorical attribute type
    • This is a good choice because using a spatial region channel, specifically with regards to a map, displays to the user where each state (categorical attribute) is. This demonstrates the expressiveness principle as a categorical attribute of the different states are represented through spatial region. Additionally, the effectiveness principle is met because positioning the different states where they go spatially on the map is the most accurate way.
  • Color hue channel (identity channel)
    • Represents categorical attribute type
    • This is a good choice because using red and blue clearly categorizes the two voting parties and the two contestants up against one another, which demonstrates the expressiveness principle as a categorical attribute is being shown with a color hue channel (an identity channel). It also shows where each states' electoral votes would have gone to, either the democrat, republican, or both. This follows the effectiveness principle with regards to being clearly discriminable with regards to if each states' electoral votes fell into the democrat, republican, or both.
  • Length channel (magnitude channel)
    • Represents ordered (quantitative, sequential) attribute type
    • This is a good choice because the different in lengths of the blue and red line follows the expressiveness principle in showing that one party (democrat) has much greater votes than the other party (republican). It also follows the effectiveness principle as it is clear as to which line is longer than the other.

Colormap(s)

  • Categorical colormap
    • Represents categorical attribute type
    • This is a good choice because the different colors (red and blue) clearly displays that there a two distinct parties and two distinct candidates (Trump and Clinton) running against each other. The two colors are perpetually distant and are roughly the same saturation and value.

Marks

  • point mark
    • Each of the points on the visualization represents a point mark that has an exit velocity and a launch angle.

Channels

  • Position on common scale-both vertical and horizontal channel (magnitude channel)
    • Represents ordered (quantitative, diverging) attribute type
    • This is a good choice because using a common scale allows for quick and easy understanding and comparison of the different batted balls with regards to their exit velocity (x-axis, horizontal positioning) and launch angle (y-axis, vertical positioning). This fulfills the expressiveness principle as it takes into account the two factors that need to be shown (exit velocity and launch angle) and does so with positioning in both directions, suggesting that there is ordering involved. This also demonstrates the effectiveness principle as the common scale provides accuracy with regards to the exit velocity and launch angle.
  • Color luminance channel (magnitude channel)
    • Represents ordered (quantitative, divergent) attribute type
    • This is not a good choice. With regards to the effectiveness principle, the color channel used is not very discriminable, specifically between the same category (below average, average, or above average). With the expressiveness principle, it does show to some extent that there is ordering involved but clearly deciphering, for instance, between two points of below average as to which one is more below average than the other is hard.
  • Color hue channel (identity channel)
    • Represents categorical attribute type
    • This is a good choice because using purple and orange to represent below and above average allows for two categories to be expressed, following the expressiveness principle. Additionally, it is easy to tell between the purple and orange, therefore, meaning that it is easy to distinguish between below and above average.

Colormap(s)

  • Divergent colormap
    • Represents ordered (quantitative, divergent) attribute type
    • This is a mediocre choice because the two colors, purple and orange, represent the extremes while the neutral color in the middle, white, represents the average. However, if another color for the average was used, the visualization may have been more effective. Even though the background of the visualization is gray, it is still very close to white and it may be hard to decipher whether there is no data in an area or if it represents the average. Also, in addition to having a color represent average, there should maybe be another color representing "no data."

Marks

  • line mark
    • Line marks are used to represent the three different age ranges.

Channels

  • Color hue channel (identity channel)
    • Represents categorical attribute type
    • This is a good choice because using blue, orange, and red clearly categorizes the people in the dataset into three distinct age categories (satisfying the expressiveness principle) and using the three different colors, you can clearly distinguish the different attrition rates for each age category (satisfying the effectiveness principle, specifically with regards to discriminability).
  • Position on common scale-vertical channel (magnitude channel)
    • Represents ordered (quantitative, sequential) attribute type
    • This is a good choice because the different of the bars along the vertical axis are of different positions suggesting difference in rate (y-axis value), which follows the expressiveness principle. With regards to the effectiveness principle, the positions that the bars are on are different enough (discriminable) to tell which age ranges have a greater or lower rate.

Colormap(s)

  • Categorical colormap
    • Represents categorical attribute type
    • This is a good choice because the different colors represent different age categories. Blue represents Middle Age group, orange represents Senior age group, and red represents Young age group.The three different colors clearly displays that there are three distinct categories in the visualization. The three colors are perpetually distant and are roughly the same saturation and value.
