Channels

  • Position-both horizontal and vertical channel (spatial region) (identity channel)
    • Represents categorical attribute type
    • This is a good choice because using a spatial region channel, specifically with regards to a map, displays to the user where each state (categorical attribute). This demonstrates the expressiveness principle as a categorical attribute of the different states are represented through spatial region. Additionally, the effectiveness principle is met because positioning the different states where they go spatially on the map is the most accurate way.
  • Color hue channel (identity channel)
    • Represents categorical attribute type
    • This is a good choice because using red and blue clearly categorizes the two voting parties and the two contestants up against one another, which demonstrates the expressiveness principle as a categorical attribute is being shown with a color hue channel (an identity channel). It also shows what each states' electoral votes would have gone to, either the democrat, republican, or both. This suggests that this follows the effectiveness principle with regards to being clearly discriminable with regards to if each states' electral votes fell into the democrat, republican, or both.

    • This is a good choice because the different colors (red and blue) clearly displays that there a two distinct parties and two distinct candidates (Trump and Clinton) running against each other.

Marks

  • point mark

Channels

  • Position on common scale-both vertical and horizontal channel (magnitude channel)
    • Represents ordered (quantitative, diverging) attribute type
    • This is a good choice because using a common scale allows for quick and easy understanding and comparison of the different batted balls with regards to their exit velocity (x-axis, horizontal positioning) and launch angle (y-axis, vertical positioning). This fulfills the expressiveness principle as it takes into account the two factors that need to be shown (exit velocity and launch angle) and does so with positioning in both directions, suggesting that there is ordering involved. This also demonstrates the effectiveness principle as the common scale provides accuracy with regards to the exit velocity and launch angle.
  • Color saturation channel (magnitude channel)
    • Represents ordered (quantitative, sequential) attribute type
    • This is a moderate-bad choice. With regards to the effectiveness principle, the color channel used is not very discriminable, specifically between the same category (below and above average). With the expressiveness principle, it does show to some extent that there is ordering involved but clearly deciphering between below average, average, and above average can be a little hard.

Colormap(s)

  • Divergent colormap
    • Represents ordered (quantitative) attribute type
    • This is a good choice because the two colors, purple and orange, represent the extremes while the neutral color in the middle, white, represents the average. However, if another color for the average was used, the visualization may have been more effective. Even though the background of the visualization is gray, it is still very close to white and it may be hard to decipher whether there is no data in an area or if it represents the average/
